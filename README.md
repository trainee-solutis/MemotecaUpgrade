# TreinamentoAngular
## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/trainee-solutis/MemotecaUpgrade.git
```

Entre no diretório do projeto

```bash
  cd MemotecaUpgrade
```

Instale todas as dependências

```bash
  npm install-all
```

Inicie o servidor

```bash
  npm start
```
